# IQOSFree Tracker

Сучасний PWA додаток для контролю звички куріння та фінансової економії.

## Особливості

- **Life Tree (Zen-механіка)**: Дерево життя, яке росте або в'яне залежно від вашої звички.
- **Розумна статистика**: Відстеження витрат та кількості використаних стіків (день/тиждень/все).
- **PWA**: Встановлюється як нативний додаток на телефон (Android/iOS).
- **Canvas Transparency**: Унікальна технологія видалення фону для ідеального вигляду на мобільних пристроях.

## Технології

- HTML5 / Vanilla CSS (with OKLCH) / Modern JS
- Tailwind CSS
- Firebase (Auth & Firestore)
- Chart.js
- Service Workers (v26)
